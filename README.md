# Library_Management_System-MJ
Spring boot Backend Library Management System Project

## Technologies Used

- **Frontend:**
  - Spring Web
  - Spring Data JPA
  - MySQL Driver
  - Lombok
    
  ## Functionality
  
  - Create a new User
  - Fetch all the Users
  - Add new book
  - Fetch all the books
  - Fetch specific book
  - Delete a book
  - Borrow a book
  - Return a book
